# Product images

This folder ships with **SVG placeholders** generated by `scripts/gen-placeholders.mjs`.

To replace a placeholder with a real product photo:

1. Drop the photo into this folder using the **same filename** as the placeholder, but with the appropriate extension (e.g. replace `jordan-1-chicago.svg` with `jordan-1-chicago.jpg` — or just keep the extension consistent).
2. If you change the file extension, update the matching `image` field in `data/shoes.json` (e.g. `"/shoes/jordan-1-chicago.jpg"`).
3. Recommended specs: 1200×1200 or larger, square, PNG / JPG / WEBP, transparent or off-white background, the shoe centred with comfortable padding.

The site uses `next/image` with `object-contain`, so any aspect ratio renders without cropping — but square images compose best in the catalog grid.

To regenerate placeholders from scratch, run:

```
node scripts/gen-placeholders.mjs
```
